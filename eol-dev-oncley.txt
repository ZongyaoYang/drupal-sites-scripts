<p><strong>Integrated Surface Flux System (ISFS) Overview</strong></p>

<p>The NCAR Integrated Surface Flux System (ISFS) is designed to study exchange processes between the atmosphere and Earthâ€™s surface. This includes the direct measurement of fluxes of momentum, sensible and latent heat, trace gases (carbon dioxide is standard) and radiation, as well as standard atmospheric and surface variables. ISFS sensors include a mix of commercial and EOL in-house designed instruments. The ISFS can be deployed as a widely distributed network of surface weather stations - measuring the surface energy budget, fluxes, and some soil parameters. Alternatively, it can be deployed in closely spaced arrays to support intensive micrometeorological research at a single site. Solar and wind power along with cellular, and satellite data transmission permit ISFS to operate in almost any location.</p>

<p style="margin-left: 40px;"><em>Configuration</em>: Ground-based, fixed<br />
<em>Number of available systems</em>: 30</p>

<h4 class="header-text-bold" style="">Request Form Questions</h4>

<p>The request form questions are to determine the EOL resources required to carry out your project.&nbsp; We are asking so that we can determine:</p>

<ul style="margin-left: 40px;">
	<li>Which sensors are required (determining the level of complexity and time to calibrate sensors).</li>
	<li>The type of tower structure needed to mount these sensors.</li>
	<li>The source of electrical power needed for each station.</li>
	<li>The type of data communication to be used at each station.</li>
	<li>How each site will be physically accessed.</li>
	<li>The amount of data processing that will be needed.</li>
	<li>The level of staffing needed.</li>
	<li>Any unusual risk factors.</li>
</ul>

<p>Assuming that the experiment design is described adequately in the Facility Request, only summary information is required in the ISFS portion of the form. If there are any questions, ISFS staff will contact the requestor.</p>

<p>If approved, ISFS staff will want to participate with the PI in a pre-project site survey, to assist in determining the optimal (scientifically and logistically) site location and to identify any unique issues.</p>

function sortTable(params) {
  var table, rows, switching, i, curr_row, next_row, curr_text, next_text, shouldSwitch;
  // get the id of the coln that was clicked
  var col = params.getAttribute('id');
  // get the current sort (asc or desc) of the clicked coln
  var sortord = params.getAttribute('data-sort');
  table = document.getElementById("project_resp");
  
  if (sortord=="999" || sortord=="desc") {
    document.getElementById(params.id).dataset.sort = "asc";
  }
  else {
    document.getElementById(params.id).dataset.sort = "desc";
  }

  switching = true;

  /*Make a loop that will continue until
  no switching has been done:*/
  while (switching) {

    //start by saying: no switching is done:
    switching = false;

    rows = table.rows;

    /*Loop through all table rows (except the
    first, which contains table headers):*/
    for (i = 1; i < (rows.length - 1); i++) {
      //start by saying there should be no switching:

      shouldSwitch = false;

      /*Get the two elements you want to compare,
      one from current row and one from the next:*/
      curr_row = rows[i].getElementsByTagName("TD")[col];
      next_row = rows[i + 1].getElementsByTagName("TD")[col];

      curr_text = curr_row.textContent.toLowerCase().trim();
      next_text = next_row.textContent.toLowerCase().trim();
      
      // check if we the coln currently set to ascending or descending sort
      if (sortord == "desc" || sortord == "999") { // if desc then sort ascending

        //check if the two rows should switch place:
        if (curr_text > next_text) {
          //if so, mark as a switch and break the loop:
          shouldSwitch = true;
          break;
        } // close if
      } // close if
      else { // if asc sort descending
     
        //check if the two rows should switch place:
        if (curr_text < next_text) {
          //if so, mark as a switch and break the loop:
          shouldSwitch = true;
          break;
        } // close if

      } // close else
    } // close for loop

    if (shouldSwitch) {
      /*If a switch has been marked, make the switch
      and mark that a switch has been done:*/
      rows[i].parentNode.insertBefore(rows[i + 1], rows[i]);
      switching = true;
    } // close if

  } // close while
}
